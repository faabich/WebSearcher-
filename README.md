# Description
Application web basée sur des recherches web avancées sur plusieurs moteurs de recherche. <br>
**Use case**: Le champ d'application est sur tout les utilisateurs souhaitant faire des recherches sur le web.

# Pré-requis
* DataBase Engine: MangoDB
* Web Framework: Reflex
* IDE used: JetBrains PyCharm
* Package manager: pip
